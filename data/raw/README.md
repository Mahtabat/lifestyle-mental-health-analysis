## Dataset Description

This project uses observational, self-reported survey data related to lifestyle habits, mental health conditions, and happiness.

The dataset includes variables such as:
- Mental health condition (e.g., anxiety, depression, PTSD, none)
- Self-reported happiness score
- Average sleep duration
- Work hours
- Screen time
- Social interaction level
- Demographic information (age group, gender, country)


## Notes
- Findings are based on observational data.
- Results indicate association, not causation.
